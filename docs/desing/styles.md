# Guía de Estilos y Patrones de Diseño

Este documento resume la identidad visual y las decisiones de interfaz de la plataforma. El diseño se ha pensado para que sea una herramienta de trabajo cómoda, directa y sin distracciones.

---

## Estética y Color (Look & Feel)

La web utiliza una base **principalmente blanca**. Esto no es casualidad: buscamos que la interfaz "respire", que no canse la vista tras mucho tiempo de uso y que las fotos de los productos sean las que manden.

### El uso del color rojo

Para romper con el blanco y guiar al usuario, utilizamos un tono **rojo oscuro/granate** como color de identidad.

- **¿Para qué se usa?:** Exclusivamente para **remarcar las cosas importantes**.
- **Intención:** Lo verás en botones de acción, estados críticos o elementos que necesitan atención inmediata. Si algo es de este color, es que el usuario tiene que interactuar con ello o es una información relevante del negocio.
- **Sensación:** Aporta un toque de seriedad y fuerza que encaja con el sector de la distribución, alejándose de colores más infantiles o puramente decorativos.

## Patrones de Diseño Web

He aplicado patrones que cualquier usuario ya conoce, para que no haga falta un manual de instrucciones:

### 1. Header Fijo (Sticky Header)

La barra de navegación siempre se queda arriba. Así, el buscador y el carrito están disponibles en todo momento, aunque la lista de productos sea eterna.

### 2. Tarjetas de Producto (Cards)

El catálogo se organiza en cartas limpias. Es la forma más fácil de ver la foto, el nombre y el precio de un vistazo antes de decidir qué añadir al carrito.

### 3. Tablas de Gestión

Para los pedidos y el stock, nada le gana a una buena tabla. Es la manera más eficiente de ver mucha información de golpe (IDs, fechas, totales) de forma ordenada y profesional.

### 4. Interacciones Suaves

Los elementos interactivos tienen pequeños cambios de tono y escala al pasar el ratón o pulsar. Son detalles sutiles para que la web se sienta "viva" y responda bien a lo que hace el cliente.
